Tomato Leaf Disease Classifier - A Student Experiment

This platform is an experimental project by Ean Kotadia, a Class 10 student exploring how computer vision models can be trained to recognize tomato plant diseases from leaf images.

Try the Classifier Demo

Project Introduction

The idea behind this project was to learn how machine learning and computer vision techniques can help in early detection of plant diseases. The classifier demonstrates how image data can be processed to generate predictions that assist in identifying common tomato leaf conditions.

Project Features

Image Recognition

Trained on example images to classify leaves as Healthy, Early Blight, Late Blight, or Leaf Mold.

Interactive Predictions

Users can upload a photo of a tomato leaf and receive a prediction and confidence score.

Educational Reports

Generates simple summaries explaining what the prediction means for plant health.

Browser-Based Deployment

Runs entirely in the browser using TensorFlow.js without requiring server infrastructure.

Methodology

The project followed this workflow:

  • Collected example images of healthy and diseased tomato leaves.
  • Trained a Teachable Machine image classification model.
  • Exported the model to TensorFlow.js format.
  • Developed a simple website to load the model and run predictions.

All steps were carried out as part of an educational science project.

Data and Tools

This project uses:

  • Example leaf photos collected by the creator and supplemented by public datasets.
  • Teachable Machine for model training.
  • TensorFlow.js for in-browser prediction.
  • HTML, CSS, and JavaScript for the web interface.

Reporting Features

The classifier can generate:

  • Predicted disease category and probability.
  • Short descriptions of each disease.
  • Downloadable example reports for demonstration purposes.

Future Plans

In the future, I plan to:

  • Collect more diverse and higher-quality image samples.
  • Experiment with more advanced model architectures.
  • Improve the user interface and visualization of results.
  • Explore deployment to mobile devices.

Disclaimer

This is an experimental educational project. The predictions are not intended for real agricultural decision-making. For professional diagnosis, please consult qualified agronomists.
