Documentation & Methodology

Learn how the Tomato Leaf Disease Classifier was developed as an independent student science project, which uses machine learning to recognize plant diseases from leaf images. This platform is intended exclusively for experimentation and learning.

Project Overview

The Tomato Leaf Disease Classifier is a self-initiated educational project created by Ean Kotadia, a Class 10 E student. The goal is to explore how computer vision models can assist in early detection of plant health issues.

  • Language: HTML, CSS, JavaScript
  • Model: Teachable Machine Image Classifier
  • Libraries: TensorFlow.js
  • Project Type: Science Project (non-commercial)
  • Status: Completed as part of Summer Vacation Homework

This project demonstrates how accessible AI tools can help students build practical solutions for agriculture.

Project Objectives

The primary objectives of this project were:

  • Understand how machine learning models are trained on labeled image data.
  • Practice collecting and categorizing example images of healthy and diseased leaves.
  • Deploy an interactive web-based classifier that predicts disease types.
  • Highlight the potential benefits and limitations of AI in agriculture.
  • Encourage other students to explore computer vision for real-world challenges.

Data Sources and Collection

The model was trained using photographs collected and organized into categories:

  • Healthy Leaves: Fresh, green tomato leaves without visible discoloration.
  • Early Blight: Leaves showing brown concentric spots and yellowing edges.
  • Late Blight: Dark lesions and rapid tissue decay.
  • Leaf Mold: Yellow patches with fuzzy growth on the underside.

Images were manually labeled and uploaded to Teachable Machine to build the classifier.

Additional sample images were used to validate the model’s predictions.

Modeling Techniques

The classifier uses Teachable Machine’s image recognition framework, which applies transfer learning on a pre-trained MobileNet architecture. This enables accurate predictions even with a small dataset.

  • Teachable Machine: Web-based tool for training and exporting TensorFlow.js models.
  • TensorFlow.js: Library to run the model directly in the browser without a backend server.
  • Client-Side Prediction: Image files are processed in the browser to classify leaf health.

The model is intentionally lightweight so it loads quickly and can be tested on most devices.

Output Features

When a user uploads or captures a photo of a tomato leaf, the system:

  • Displays the predicted class label (e.g., Healthy, Early Blight, Late Blight, Leaf Mold).
  • Shows the confidence score for each prediction.
  • Provides simple guidance on what the result means.

All predictions are for demonstration and learning purposes only.

Limitations and Disclaimer

This project is a student experiment and has not been validated by agricultural experts. It should not be used for actual diagnosis or treatment decisions.

Model accuracy depends on image quality, lighting, and dataset size. For professional advice, consult qualified agronomists.
