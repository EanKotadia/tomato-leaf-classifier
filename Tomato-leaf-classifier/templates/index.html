Tomato Leaf Disease Classifier

This project showcases an AI-powered image recognition system for identifying common tomato leaf diseases. Developed by Ean Kotadia, Class 10 E, as part of the Science Summer Vacation Homework.

Project Overview

The classifier was trained using Teachable Machine and distinguishes between healthy leaves and common diseases, including Early Blight, Late Blight, and Leaf Mold.

This project demonstrates how computer vision and machine learning can support agriculture by enabling early detection of plant health issues.

How It Works

  • Step 1: Collect images of healthy and diseased leaves.
  • Step 2: Train an image classification model with Teachable Machine.
  • Step 3: Export or link the trained model for use in this web interface.
  • Step 4: Use TensorFlow.js to run predictions directly in your browser.
